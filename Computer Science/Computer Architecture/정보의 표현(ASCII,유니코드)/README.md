# 문자의 표현
## ASCII
문자를 숫자로 표현할 수 있도록 정해진 약속이 ASCII코드이다.   
128개의 부호로 정의되어있는데 A는 65, B는 66이다.   
문자인 0은 48, 9는 57이고,   
소문자 a는 97, z는 112이다.   
65는 2진법으로 1000001이다.   
   
## 유니코드
