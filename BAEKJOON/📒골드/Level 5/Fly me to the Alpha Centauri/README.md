짝수와 홀수로 나뉘어진다.   
   
10회 이동할때 최대 거리 30이 
(n/2) * (n/2)+1이 된다.   
5 * 6   
다시 만약 12회면   
6 * 7가 된다.   
    
11회면 최대거리 36으로   
(n/2)+1 * (n/2)+1   
6*6이 되고,   
13회면 7*7이 된다.   
(n/2)를 num으로 계산하고,   
찾았을때 짝수쪽에서 찾았으면 n*2를, 홀수쪽에서 찾았다면 (n*2)+1 을 해준다.   

```
import Foundation
func solution(){
    let t = Int(readLine()!)!
    for _ in 0..<t {
        let input = readLine()!.split(separator: " ").map{Int(String($0))!}
        let diff = input[1] - input[0]
        var n = 0
        
        while true {
            if diff <= n * (n+1) {
                print(n*2)
                break
            } else if diff <= (n+1) * (n+1) {
                print((2*n)+1)
                break
            } else {
                n += 1
            }
        }
        
    }
}
solution()
```
